# Features

This page is to test some of the features of `mkdocs` and the `material` theme.

## Syntax Highlighting

For example the syntax highlighting of code snippets.
A code snippet looks like this:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello World")
}
```
